Data flow principle

Zero data architecture
No backend. No server. No cloud. No account. No analytics. No crash reporting. No third-party services of any kind. All user data stays in on-device SQLite — permanently. Apple StoreKit handles payments; developer never sees payment data. The only external URL the app references is the privacy policy page at mukuma.com.
